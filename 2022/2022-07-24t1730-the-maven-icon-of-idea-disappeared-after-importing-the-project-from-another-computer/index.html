从另一台电脑导入项目到IDEA后报错 点击Maven图标立即消失

Written by sinowrt with ♥ on July 24, 2022 in IDEA

0x00 报错信息

Cannot run program “XXX” (in directory “XXX”): CreateProcess error=2, 系统找不到指定的文件。

0x01 报错原因

该项目是在别的电脑压缩打包拷贝过来的，所以项目的Java编译器路径、Maven路径以及Maven依赖的Jave环境路径都是原主机的

0x02 解决办法

点击IDEA右上角的齿轮，Project Structures - Project - SDK，将SDK配置为本机已安装的SDK路径，接着打开Project下面的Modules，将Module SDK也配置好。

Maven图标消失了的话，点击菜单 Help - Find Action, 搜索 Maven Settings， 点进去，Maven home path、 setting file、 repository都根据本机具体情况进行配置； 回到左侧列表树，展开Maven - Runner，将JRE也配置一下

最后确认保存即可。
