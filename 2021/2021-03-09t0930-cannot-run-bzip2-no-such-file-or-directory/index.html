如何解决cannot run bzip2: No such file or directory错误

Written by sinowrt with ♥ on March 9, 2021 in linux

0x00 错误描述

今天在使用tar解压文件的时候，报了一个错 cannot run bzip2: No such file or directory

0x01 解决方法

这个错误是因为系统中缺少bzip2，使用yum安装上就好了 yum -y install bzip2
