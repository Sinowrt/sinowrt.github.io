怎样查看linux版本

Written by sinowrt with ♥ on March 8, 2021 in linux

0x00 相关命令

# 可显示电脑以及操作系统的相关信息
uname -a

# 查看正在运行的内核版本
cat /proc/version

# 查看发行版本信息
cat /etc/issue

# 适用于所有的linux，包括Redhat、SuSE、Debian等发行版，但是在debian下要安装lsb
lsb_release -a
